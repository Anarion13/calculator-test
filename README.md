# Web Calculator

A lightweight browser-based calculator with the core features you expect from a first implementation:

- addition, subtraction, multiplication, and division
- decimal support
- sign toggle and percentage conversion
- clear and keyboard input support
- small automated test suite for the calculator engine

## Run locally

Because this project is a static web app, you can open `index.html` directly in a browser or serve the folder with a simple local server:

```bash
python3 -m http.server 8000
```

Then open `http://localhost:8000`.

## Run tests

```bash
npm test
```
